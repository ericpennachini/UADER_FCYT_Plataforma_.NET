IMPORTANTE!!

Archivo 'App.config': En el tag <connectionStrings> y <add ... /> se encuentra el string que se usa para la conexión con la BD.  

Cada uno, cuando va a ejecutar esta aplicación, previamente tiene que tener creada una BD en su servidor local. 
Luego, hay que modificar esta entrada para que tome la BD local de cada uno (servidor y nombre de la BD).

Modificar:
	'data source= [SERVIDOR]'
	'initial catalog= [NOMBRE DE LA BASE DE DATOS]'

{Para hacerlo más rápido, CTRL+F y buscar 'data source' e 'initial catalog'}

El resto no se tendria que tocar, salvo que usen un servidor remoto con usuario y contraseña, que no sería el caso porque estamos usando el servidor local de la máquina.

Tambien hay que modificar el archivo sql para que use la base de datos local de cada uno para ejecutarse. Buscar:

	SET QUOTED_IDENTIFIER OFF;
	GO
	USE [BASE_DE_DATOS]; // modificar el texto entre corchetes con el nombre de la BD de cada uno
	GO
	IF SCHEMA_ID(N'dbo') IS NULL EXECUTE(N'CREATE SCHEMA [dbo]');
	GO
